How the portfolio automation works

The workflow wakes up on a schedule, reads your ticker list, collects recent market headlines, asks an AI model to summarize sentiment, then sends a compact Telegram alert and a polished HTML email.

Animated Automation Flow

1. Schedule Trigger

n8n starts the automation every hour when the workflow is active.

Every Hour

2. Prepare Portfolio

The ticker string is loaded from an environment variable and split into individual stocks.

PORTFOLIO_TICKERS

3. Fetch News

For each ticker, the workflow searches Google News RSS for fresh stock headlines.

RSS Feed Read

4. Clean + Group

News items are normalized, aggregated by ticker, sorted by date, and trimmed to the latest articles.

Aggregate + Code

5. AI Analysis

Groq generates a portfolio overview plus bullish, bearish, or neutral sentiment for each stock.

Groq Model

6. Deliver Updates

A short Telegram message and a designed HTML email are sent to the configured recipients.

Telegram + Gmail

Runtime Setup

After importing the workflow into n8n, reconnect Gmail, Telegram, and Groq credentials. Recipients are kept in environment variables so the repository remains safe to share publicly.

GMAIL_SEND_TO=you@example.com TELEGRAM_CHAT_ID=123456789 PORTFOLIO_TICKERS=ZENTEC, RAILTEL, DIXONTECH
